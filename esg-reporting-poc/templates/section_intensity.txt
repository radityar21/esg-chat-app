### REQUIRED ELEMENTS
1. Opening paragraph: state both intensity metrics with units
   - Revenue intensity: tCO2e per IDR billion
   - FTE intensity: tCO2e per FTE
2. Methodology paragraph: explain denominators (revenue source, FTE definition)
3. Scope coverage statement:
   - Revenue intensity: Scope 1 + 2 (market-based) + 3 Category 15 gross
   - FTE intensity: Scope 1 + 2 (market-based) ONLY (operational emissions)
4. Intensity calculation table: numerator, denominator, ratio for both metrics
5. Year-over-year comparison (OMIT if prior_year values are NULL)
6. Contextual interpretation: explain whether intensity improved/worsened and why
   (e.g., revenue growth outpaced emissions growth → intensity improved)
7. Peer comparison statement (INCLUDE if framework = IFRS_S2 or CSRD_ESRS_E1):
   reference industry benchmarks if available

### CALCULATION CONSTRAINT
intensity_tco2e_per_idr_bn = total_tco2e / revenue_idr_billion
intensity_tco2e_per_fte = scope1_plus_scope2_tco2e / fte_count
Both must reconcile with stated numerators and denominators in narrative.

### FRAMEWORK-SPECIFIC NOTES
- GRI 305-4: Requires organization-specific metric (denominator) justification
- IFRS S2: Requires industry-specific intensity metrics per SASB guidance
- CSRD E1-6: Requires revenue-based intensity; FTE optional but recommended
- OJK PSPK: Requires revenue-based intensity for financial institutions
