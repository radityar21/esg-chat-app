## SECTION TITLE (MANDATORY)
title: "1.5 Emission Reduction Initiatives & Targets"

## FRAMEWORKS ADDRESSED
- GRI 305-5 — Reduction of GHG emissions
- IFRS S2 Paragraph 33-36 — Climate-related targets and transition planning
- ESRS E1-4 DR E1-4 — GHG emission reduction targets & decarbonization levers
- OJK PSPK — Target Pengurangan Emisi GRK

## DATA INPUT
Uses metrics from ghg_summary:
- scope1_tco2e, prior_year_scope1_tco2e, yoy_change_pct
- scope2_location_tco2e, prior_year_scope2_location
- vs_base_year_change_pct (if available)

## CONDITIONAL DISCLOSURE LOGIC

IF yoy_change values are available AND base_year data exists:
  → Generate FULL reduction disclosure:
    - YoY changes per scope
    - vs Base year comparison
    - Attribution: initiatives vs organic change
    - Reduction initiative inventory from RAG benchmarks

IF yoy_change available BUT no formal reduction targets in DATA INPUT:
  → Generate PARTIAL disclosure + GAP ANALYSIS:
    - Report observed changes (increase or decrease)
    - Acknowledge: "structured reduction program not yet formalized"
    - Provide peer benchmarks (BRI EV fleet, DBS RE100, BCA green building)
    - Recommend specific reduction initiatives

IF ALL reduction data is NULL:
  → Generate FULL GAP ANALYSIS:
    - State GRI 305-5 + IFRS S2 Para 33 requirements
    - Peer comparison: what other banks have achieved
    - Roadmap to establish reduction tracking
    - Priority matrix
